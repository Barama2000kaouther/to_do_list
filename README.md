# Django Todo App

A simple Todo List application built with Django, HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Update tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Active)

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
